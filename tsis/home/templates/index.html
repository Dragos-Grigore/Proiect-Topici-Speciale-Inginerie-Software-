Lens-AI

This application uses the Lens-ContextualAI, which is an approach for tackling computer vision problems by leveraging the power of large language models (LLMs) as stated by the paper: "Towards Language Models That Can See: Computer Vision Through the LENS of Natural Language".

It highly represents the synergy between Natural Language Processing and Computer Vision through a State-of-the-Art model able to recognise what a certain picture is about.

How to use it?

Navigate to the "Playground" tab, then click on "Browse Image" and select the desired image, choose an LLM from the dropdown menu, and lastly write a query for the model and click the "Try it" button in order to get a response from the system.
